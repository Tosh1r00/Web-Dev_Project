# Web-Dev_Project

## Project description
Our project is an e-commerce for the sale of online gaming products.

What does it include?
1) Login page:
-Client can register (Username + password)
-If client is already registered, -> can be reathurised (and go to main page after)

Additional:
-Forgot password [button]?
-Block with description and interactive content
-navbar with links to other pages

2) Main page:
-On the main page will be product-card (with field)
-Sorting bar (Different types of sorting)
-Categories
-trade section

*If we have time we will add read-only section FAQ*
Additional:
- Review / Comments section
- Balance block
- Link [button] to profile (user page)

3) User page:
 - Profile info
 - Inventory (available/bought products)
 - Button fullfill the balance
 - Transaction story

Additionally:
-Button to sell the product back with -30% from the initial price
